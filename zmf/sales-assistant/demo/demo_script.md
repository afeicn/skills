# 销售助手 Demo Script

## 演示目标

展示用户通过飞书向“销售助手”提交任务后，数字员工如何识别场景、调用 Hermes Agent skills、生成结构化结果、提交人工确认并归档。

## 演示输入

```text
@销售助手 根据这段客户拜访记录，生成客户画像、需求摘要和下一步跟进建议。
```

## 演示流程

1. 飞书发送演示指令和脱敏附件。
2. 数字员工返回任务受理卡，说明识别到的场景包和所需材料。
3. 调用 2-3 个核心领域 skills 生成结构化结果。
4. 输出风险/缺口和需人工确认事项。
5. 生成确认卡片，模拟负责人确认。
6. 归档结果并返回归档记录。

## 演示兜底

- 准备一份静态结果页。
- 准备一段录屏备份。
- 网络或接口不稳定时切换演示模式 `C18_demo_mode`。
