# 标书助手 P0 演示测试样例

## 样例 1：招标文件解析

用户飞书输入：

```text
@标书助手 帮我分析这个示范项目招标文件，重点提取资格条件、评分项和截止时间。
```

附件：

- demo_tender_document.pdf

期望流程：

1. 解析招标文件。
2. 提取项目摘要。
3. 提取资格条件。
4. 提取评分标准。
5. 提取提交要求和废标风险。
6. 输出招标文件解析报告。

期望输出摘要：

```text
处理结论：需人工确认是否投标。
已提取：资格条件、评分标准、提交要求、风险提示。
下一步：建议由商务负责人确认资质和投标意向。
```

## 样例 2：投标任务拆解

用户飞书输入：

```text
@标书助手 根据刚才的招标文件，帮我拆投标任务。
```

期望输出：

```text
任务清单：
1. 商务材料准备
2. 技术方案撰写
3. 项目案例整理
4. 报价确认
5. 标书审查
6. 盖章和提交
```

## 样例 3：标书审查

用户飞书输入：

```text
@标书助手 帮我审查这个投标文件草稿有没有缺项和偏离风险。
```
