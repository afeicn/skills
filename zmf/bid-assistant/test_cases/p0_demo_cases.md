# 标书助手 P0 演示测试样例

## 样例 1：评分模型与结构设计

用户飞书输入：

```text
@小招 帮我分析这个示范项目招标文件和作业设计，建立评分模型，并生成服务方案结构。
```

附件：

- demo_tender_document.pdf
- demo_design_document.pdf

期望流程：

1. 解析招标文件。
2. 检查作业设计关键数字。
3. 提取资格条件和评分标准。
4. 建立评分模型。
5. 生成无负偏离索引和证据链目录。
6. 生成服务方案十二章结构。

期望输出摘要：

```text
处理结论：需人工确认是否投标。
已生成：评分模型、客观分材料清单、无负偏离索引、服务方案结构。
下一步：建议由商务负责人确认资质、投标意向、报价和商务承诺。
```
