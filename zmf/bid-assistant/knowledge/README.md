# 标书助手知识库目录

本目录用于存放标书助手运行所需材料。当前先放说明，后续由商务或标书负责人补充真实文件。

## 必需知识包

1. 公司资质材料
2. 人员证书与团队介绍
3. 项目案例库
4. 历史标书库
5. 标书模板库
6. 技术方案素材库
7. 商务承诺与合同条款素材
8. 常见废标风险清单
9. 投标复盘记录

## 文件命名建议

```text
company_公司资质清单.md
case_项目案例库.md
template_技术方案模板.md
template_商务响应模板.md
risk_废标风险清单.md
retro_投标复盘记录.md
```

## 重要要求

- 资质、业绩、人员信息必须真实可核验。
- 报价和商务承诺必须由负责人确认。
- 演示样例使用脱敏或虚拟示范项目，不使用真实客户敏感信息。
