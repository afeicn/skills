# 财务助手 Workflows

## 一、报销申请受理

目标：将员工通过飞书提交的发票和报销说明，转化为可审核的报销单草稿。

```text
触发：员工向“财务助手”发送发票图片/PDF + 报销说明

步骤：
1. 接收飞书消息和附件
2. 识别用户身份和部门
3. 判断任务类型是否为报销申请
4. 对发票进行 OCR 和字段抽取
5. 判断费用类型
6. 检查是否缺少必要材料
7. 生成报销单草稿
8. 请员工确认或补充材料

输出：
- 报销单草稿
- 缺失材料清单
- 员工确认卡片
```

## 二、报销单审核

目标：对报销单草稿进行财务初审，生成审核意见。

```text
触发：员工确认报销单草稿，或财务人员主动要求审核

步骤：
1. 发票验真或读取演示验真结果
2. 检查历史台账是否疑似重复报销
3. 检查费用类型是否匹配制度
4. 检查附件完整性
5. 判断项目、部门、成本中心归属是否明确
6. 检查预算和费用科目，如预算数据可用
7. 判断审批权限和审批路径
8. 生成审核意见

输出：
- 通过建议
- 需补充材料
- 需人工复核
- 疑似不合规
```

## 三、审批流转

目标：将审核后的报销事项提交给正确的主管或财务复核人。

```text
触发：报销审核意见为“通过建议”或“需主管确认”

步骤：
1. 根据部门、金额、费用类型匹配审批路径
2. 生成飞书审批卡片草稿
3. 发送给员工进行最终确认
4. 员工确认后提交主管
5. 跟踪审批状态
6. 如被退回，生成补充材料通知
7. 审批完成后归档

输出：
- 飞书审批卡片
- 审批状态
- 退回补充通知
- 归档记录
```

## 四、财务制度问答

目标：回答员工关于报销流程、材料、费用标准、审批路径的问题。

```text
触发：员工向财务助手提问

步骤：
1. 判断问题类型
2. 检索财务制度和 FAQ
3. 引用制度依据
4. 输出流程说明、材料清单或标准
5. 如问题涉及特殊情况，建议咨询财务人员

输出：
- 制度问答
- 流程指引
- 所需材料清单
```

## 五、台账与费用分析

目标：将已审批或已审核的报销事项沉淀为财务台账，并支持费用分析。

```text
触发：审批完成或财务人员要求更新台账

步骤：
1. 读取报销单和审核记录
2. 提取报销人、部门、项目、费用类型、金额、日期
3. 写入台账草稿
4. 标记审批状态和归档文件位置
5. 按月、部门、项目、费用类型生成统计
6. 提示异常费用或高频问题

输出：
- 报销台账记录
- 月度费用摘要
- 异常费用提示
```
