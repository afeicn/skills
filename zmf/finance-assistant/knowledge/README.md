# 财务助手知识库目录

本目录用于存放财务助手运行所需知识材料。当前先放说明，后续由财务负责人补充真实文件。

## 必需知识包

1. 财务报销制度
2. 差旅费标准
3. 交通费、住宿费、餐补、招待费标准
4. 发票与附件样例
5. 报销单模板
6. 审批权限规则
7. 项目和部门成本中心清单
8. 历史报销台账样例
9. 常见问题 FAQ

## 文件命名建议

```text
policy_财务报销制度.md
policy_差旅费标准.md
template_报销单.xlsx
sample_发票样例/
ledger_历史报销台账.xlsx
faq_财务制度问答.md
```

## 重要要求

- 真实发票、身份证、银行卡、薪酬等敏感信息不得用于公开演示。
- 演示样例应使用脱敏数据或虚拟数据。
- 制度版本需要标注生效日期。
