# Hermes Skills by afeicn

Personal skills for [Hermes Agent](https://hermes-agent.nousresearch.com/).

## Install

```bash
# Add this repo as a skill tap
hermes skills tap add afeicn/skills

# Then install any skill
hermes skills install moji-daily
```

## Skills

| Skill | Description |
|-------|-------------|
| [moji-daily](productivity/moji-daily/) | 墨记 — 每日任务追踪系统。通过聊天对话自动管理每日待办事项 |
| [my-novel-writer](my-novel-writer/) | 长篇小说创作助手，支持人物、世界观、大纲、章节生成与项目记忆 |
| [novel-writing](novel-writing/) | 长篇网文创作技能包，用于文风一致、设定管理、伏笔追踪与章节质量控制 |
| [humanizer](humanizer/) | 去除 AI 写作痕迹，让文本更自然、更像真人写作 |
| [txt-to-poster-video](txt-to-poster-video/) | End-to-end workflow: narration script → vertical short video with illustrations, cover, end card, voiceover & captions |
| [video-script-generator](video-script-generator/) | 根据视频思路/大纲生成口播稿，自然对话风格 |
| [poster-style-generator](poster-style-generator/) | 海报风格生成器 — vertical cover & end card with gpt-image-2 |
| [video-illustration-generator](video-illustration-generator/) | Generate ordered body illustration frames for short-video scripts |
| [pic-to-video-generator](pic-to-video-generator/) | Image序列 + 口播稿 → voiceover + captions + 9:16 MP4 |
