# Hermes Skills by afeicn

Personal skills for [Hermes Agent](https://hermes-agent.nousresearch.com/).

## Install

```bash
# Add this repo as a skill tap
hermes skills tap add afeicn/skills

# Then install any skill
hermes skills install moji-daily
```

## Skills

| Skill | Description |
|-------|-------------|
| [moji-daily](productivity/moji-daily/) | 墨记 — 每日任务追踪系统。通过聊天对话自动管理每日待办事项 |
