# 项目管理助手 Runbook

## 部署前检查

1. 确认部署环境可运行 Hermes Agent。
2. 确认飞书机器人可用。
3. 确认 `employee.yaml` 中的 `hermes_agent_id` 为 `project_management_assistant`。
4. 将知识库、模板、演示样例放入对应目录。
5. 配置权限、审计日志、归档目录和人工确认人。

## 最小演示能力

- 飞书消息接收与路由；
- 文件或文本解析；
- 至少 2 个领域 skills；
- 结构化输出；
- 人工确认提示；
- 结果归档。

## 生产化待办

- 对接真实业务系统或台账；
- 配置权限和审计日志；
- 补齐正式知识库和模板；
- 增加失败兜底和转人工队列；
- 每周复盘 skill 命中率、退回率和人工修改点。
